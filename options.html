'Save code' button for ChatGPT

Adds a seamless 'Save code' button next to the 'Copy code' button within every ChatGPT code box. Many file extensions are recognized automatically. If any are missing for your use case, just add/edit them below.

How to use:

  1. Add pair: Add a new extension pair. Fill in the code language as shown in the ChatGPT code box and its corresponding file extension.
  2. Remove: Remove a pair.
  3. Save: Save your changes. Changes should be immediate even on already opened ChatGPT tabs.
  4. Defaults: Reset the language-extension pairs to the defaults. First click 'Defaults', then 'Save'.

For example, if you want the plugin to recognize "javascript" files with ".js" extension, type "javascript" in the programming language box and ".js" in the extension box, then click Save. You can add multiple pairs with the same extension if needed.
